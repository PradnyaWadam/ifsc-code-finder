# ifsc-code-finder
The IFSC Code Finder is a web-based project developed using PHP and MySQL that helps users easily find the IFSC code of any bank branch online. It has two main modules — User and Admin. Users can search for IFSC codes by entering the bank name, branch name, or PIN code without the need to visit the bank physically. The Admin module allows complete control over the website, including managing banks, states, cities, and branch details through a user-friendly dashboard. Admins can also add, edit, or delete records, update their profile, and recover passwords. This project provides all necessary files including HTML, CSS, JavaScript, and PHP code, and can be customized easily for personal or client use.
